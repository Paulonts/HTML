Entendimento sobre tags (aula 1)

Nessa aula foi aprendido sobre umas das principais tags do html como:

  • strong> usada para deixar a palavra em negrito.
  • id="" utilizada para localizar um objeto ou string.
  • style="" utilizada para modificar visualmente um texto
  • class="" utilizada para criar uma classe, onde tudo que estiver
    dentro daquela classe vai ser modificada.
  • input type="" nesse temos 3 tipos de input: text, number e color
    input type="text" abre um espaço onde voce digite uma mensagem do tipo texto
    input type="number" abre um espaço onde voce digite uma mensagem do tipo número
    input type="color" abre um espaço onde voce seleciona uma cor, e é passado a coordenada daquela cor.
  • img width="" src="" esse comando serve para você colocar uma imagem no so seu html.
    width: usado para mudar o tamanho da sua imagem
    src: colocar o link da imagem, para você poder visualizar a imagem no seu html.

Linkamento e seus tipos (aula 4)

Nessa aula foi ensinado a como fazer linkamento na web ou no próprio site.

  • < a href="" title="" target=""> titulo< /a> esse é o esoço para você conseguir fazer linkamento
    href -- dentro das aspas duplas você coloca o link da pagina ou arquivo html que voce quer ir
    html tem que estar no mesmo projeto já o link de algum site pode ser colocado diretamente
    title -- ao passar o mouse em cima da palavra vai ter um texto.
    target -- você escolhe o tipo de linkamento que você deseja temos os:
    "_blank" vai abrir uma nova pagina.
    "_self" -- vai abrir uma pagina na pagina que você está.
